# Description
Productivity Tracker is a Chrome Extension built using React, Vite, Tailwind CSS, and Manifest V3 that helps users monitor and analyze their browsing habits. It tracks active website usage in real time, categorizes websites as productive, unproductive, or neutral, and stores analytics locally using Chrome storage APIs. The extension includes a popup interface for quick daily insights and a dashboard with charts, weekly reports, top websites, and productivity trends. It also supports idle detection, tab switching detection, and session recovery for accurate tracking. The project focuses on clean architecture, practical browser tracking, and a polished analytics experience without requiring a backend server.
